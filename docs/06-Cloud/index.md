# Cloud

Seccion orientada a Azure y decisiones de plataforma para backend, integraciones y despliegues empresariales.

## Subareas

- `Azure`: servicios de computo y mensajeria.
- `Identity`: identidad, autenticacion de workloads y secretos.
- `Storage`: almacenamiento gestionado.
- `Architecture`: criterios de diseno cloud.

## Enfoque recomendado

- Entender servicio, caso de uso y trade-off.
- Saber justificar cuando usar servicio gestionado vs contenedores.
- Relacionar seguridad, costo, operacion y velocidad de entrega.
