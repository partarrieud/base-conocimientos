# Proyectos Reales

Seccion pensada para documentar proyectos, decisiones, incidentes y aprendizajes con formato profesional reutilizable.

## Subcarpetas

- `Project-Templates`: referencias de estructura para documentar proyectos.
- `Projects`: ejemplos de casos reales o realistas listos para adaptar.
- `Postmortems`: incidentes documentados con foco en aprendizaje.

## Como usar esta carpeta

- Adaptar los ejemplos a experiencia propia.
- Completar metricas, resultados e impacto real.
- Reutilizar estos documentos para preparar historias STAR y respuestas de entrevistas.
