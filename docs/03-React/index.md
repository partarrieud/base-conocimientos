# React

Seccion orientada a entrevistas donde esperan que puedas conversar de frontend con criterio, especialmente integracion con APIs REST, manejo de estado y performance.

## Subareas

- `Basics`: hooks y estado.
- `Advanced`: rendering, performance y SSR.
- `Architecture`: diseno de componentes y decisiones de frontend.

## Enfoque recomendado

- Entender como renderiza React.
- Saber cuando un problema es de estado, arquitectura o performance.
- Poder comparar SPA y SSR sin respuestas genericas.
