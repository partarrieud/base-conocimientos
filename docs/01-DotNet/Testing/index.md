# Testing en .NET

Esta subcarpeta queda preparada para profundizar testing de aplicaciones ASP.NET Core, integracion con bases de datos, pruebas de contrato y estrategias de automatizacion.

## Foco recomendado

- pruebas unitarias sobre logica de dominio,
- pruebas de integracion sobre repositorios y APIs,
- criterios para evitar tests fragiles,
- uso responsable de mocks.
