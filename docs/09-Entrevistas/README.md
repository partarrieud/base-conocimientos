# Entrevistas

Seccion orientada a practicar respuestas para entrevistas Senior, Lead y Architect con foco en el stack principal del repositorio.

## Subcarpetas

- `DotNet`: preguntas para Senior .NET.
- `SQL`: preguntas para Senior SQL Server.
- `React`: preguntas para Senior React.
- `Arquitectura`: preguntas para Software Architect.
- `SystemDesign`: ejercicios y preguntas de diseno.
- `Leadership`: preguntas para Tech Lead y liderazgo tecnico.
- `StudyPlans`: planes de repaso y simulacion.

## Como usar esta carpeta

- Leer primero la pregunta.
- Intentar responderla en voz alta en 2 minutos.
- Comparar con la respuesta esperada y la respuesta modelo.
- Ajustar con ejemplos propios de proyectos reales.
