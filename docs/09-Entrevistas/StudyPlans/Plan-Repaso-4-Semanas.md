# Plan de Repaso de 4 Semanas

Plan corto pensado para preparar entrevistas Senior/Lead con foco en .NET Backend, SQL Server, React, Arquitectura, Azure, Kubernetes y Kafka.

## Semana 1

- .NET: Dependency Injection, Async/Await, Middleware, EF Core, Dapper.
- SQL: Indexes, Execution Plans, Transactions, Isolation Levels.
- Practica: responder 10 preguntas de `09-Entrevistas/DotNet` y `09-Entrevistas/SQL`.

## Semana 2

- React: Hooks, State Management, Performance Optimization, SPA vs SSR.
- Testing: Unit, Integration, Mocking, Test Pyramid.
- Practica: respuestas de React y ejemplos reales de proyectos.

## Semana 3

- Arquitectura: SOLID, DDD, Clean, Hexagonal, Microservices, Outbox.
- Kafka: Fundamentals, Consumers, Producers, Consumer Groups, Idempotency.
- Practica: simulacro de preguntas Tech Lead y Software Architect.

## Semana 4

- Azure: App Service, Functions, Service Bus, Managed Identity, Key Vault, AKS.
- Infraestructura: Docker, Kubernetes, Deployments, Ingress, Observability.
- Practica: dos ejercicios de System Design y repaso de experiencias reales.

## Regla de cierre

Cada semana deberia terminar con:

- 1 simulacro oral,
- 1 documento propio actualizado,
- 1 lista de huecos tecnicos detectados,
- 1 historia real preparada con formato STAR.
