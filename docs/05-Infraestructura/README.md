# Infraestructura

Seccion dedicada a contenedores, orquestacion, delivery continuo y observabilidad.

## Subareas

- `Containers`: Docker e imagenes.
- `Orchestration`: Kubernetes y despliegue.
- `CI-CD`: pipelines y automatizacion.
- `Observability`: logs, metrics y trazas.

## Enfoque recomendado

- Poder explicar desde build hasta despliegue y monitoreo.
- Entender fallos comunes de contenedores y Kubernetes.
- Relacionar infraestructura con operacion real y no solo con YAML.
